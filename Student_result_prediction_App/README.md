# 📚 Student Result Prediction Web App

A Machine Learning based web application that predicts whether a student will **PASS or FAIL** based on the number of hours studied.

This project was built during the **ML Pipeline & Voice Assistant Workshop**, where I secured **2nd Rank**.

---

## 🚀 Project Overview

This project demonstrates how a machine learning model can be integrated into a real-world application using **Flask API** and **Streamlit UI**.

The application allows users to enter the number of hours studied and instantly get a prediction of whether the student will pass or fail.

---

## 🧠 Features

- Machine Learning model trained using **Logistic Regression**
- **Data preprocessing** and dataset handling
- **Flask API** for model prediction
- **Streamlit UI** for interactive user input
- Instant prediction results

---

## ⚙️ Installation & Setup

Move into the project folder:
cd Student_result_prediction_App
Install dependencies:
pip install -r requirements.txt
---

## ▶️ Run the Project

### Run Model Training
python model_training.py

### Run Flask API
Python app.py

### Run Streamlit App
streamlit run streamlit.py
